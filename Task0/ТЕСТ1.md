# Какова основная цель документирования при программной разработке?
a) Уменьшение количества разработчиков
b) Управление технологическим процессом и передача информации
c) Сокращение времени разработки

**Ответ: b**

# Какие типы документации используются в процессе разработки?
a) Только технологическая документация
b) Только эксплуатационная документация
c) И технологическая, и эксплуатационная документация

**Ответ: c**

# Какой принцип лежит в основе стратегии документирования?
a) Стремление к сокращению количества документов
b) Охват жизненным циклом разработки
c) Непрерывное обновление документации

**Ответ: b**

# Какие артефакты являются частью документации разработки?
a) Требования, проектные спецификации, порядок контроля
b) Бухгалтерская отчетность, календарный план
c) Различные рабочие статьи

**Ответ: a**

# Какие типы документов входят в эксплуатационную документацию?
a) Технологические инструкции, руководство пользователя, брошюры
b) Пользовательские отзывы, технические спецификации, исходный код
c) Журналы, эссе, рефераты

**Ответ: a**

# Какие принципы следует учитывать при создании технической документации?
a) Избегать использования графиков и схем
b) Соответствовать читательской аудитории
c) Создавать документы только после завершения проекта

**Ответ: b**
